# 🎧 Lofi Focus Journey: Pomodoro Timer Minimalista

Uma interface de foco imersiva que utiliza a técnica Pomodoro, integrada com uma estética Lofi e fundo em vídeo para máxima concentração.

---

## 🎨 Prévia
![Lofi Focus Preview](video.mp4) 
> *Interface limpa com timer regressivo, citações motivacionais e fundo dinâmico.*

## ✨ Destaques
* **Técnica Pomodoro:** Timer configurado para ciclos de 25 minutos de foco total.
* **Estética Imersiva:** Background em vídeo com sobreposição (overlay) para não cansar a vista.
* **Design Minimalista:** Relógio flutuante com efeito de vidro (glassmorphism) e desfoque de fundo.
* **Citações Inspiradoras:** Frases dinâmicas para manter a motivação durante a jornada.
* **Leve e Rápido:** Construído com Vanilla JS, garantindo zero atraso no timer.

## 🛠️ Tecnologias
* **HTML5** (Vídeo nativo e estrutura)
* **CSS3** (Posicionamento absoluto, Backdrop-filter, Flexbox)
* **JavaScript** (Lógica de contagem regressiva e manipulação de DOM)
